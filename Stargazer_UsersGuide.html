S E C T I O N . 1 ---- Controlling the telescope remotely using AutoStar Suite


1.1 Server-Side

These steps are to be done on the computer that is connected directly to the telescope.

  1. Turn on the telescope and the computer it is connected to.

  2. Start the Net Scope Server
    1. Open Net Scope Server. Start > Programs > Meade > Net Scope Server
    2. A small window will appear. Take note of the Scope Port, Dome Port and Network Port (by default they are 1, 2 and 5401).
    3. Click on the Set Password button and enter a password of your choice. This is the password users will need to enter when they try to use the telescope from a different computer over the internet. Click OK
    4. Check the box beside Listen. The Status field should now display: Listening on port XXXX (where XXXX is the number provided in the Network Port field above).
    5. BE SURE TO LEAVE THE TELESCOPE ON.
    6. BE SURE TO LEAVE THE COMPUTER ON.
    7. DO NOT EXIT THE NET SCOPE SERVER WINDOW.

1.2 Client-Side

  1. Turn on the computer.

  2. Make sure AutoStar Suite is installed on the computer. The program is available for free from the Meade website. As of writing this guide, AutoStar Suite Version 5.5 was the newest avaibable version. Download and install it. http://www.meade.com/support/downloads.html

  3. Open AutoStar Suite. Start > Programs > Meade > AutoStar Suite

  4. Connect to the Net Scope Server
    1. Telescope > Protocol > AutoStar via Network
    2. Enter the IP Address and Network Port of the server (the computer, mentioned in above in Section 1.1, which is directly connected to the telescope).
      --- seek additional help if you dont know the ip address ---
    3. Click Connect
    4. Enter the password set in Section 1.1
    5. A large WARNING message will appear. Don't worry, this is normal.
    6. AutoStar is now able to control the telescope over the internet.
